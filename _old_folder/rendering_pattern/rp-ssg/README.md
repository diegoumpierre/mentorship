# SSG - Static Site Generation

## How to run

```bash
npm run build
open dist/index.html
```

## What to notice

- The timestamp is frozen — it was set when you ran the build
- Refresh the page as many times as you want, the timestamp never changes
- Run `npm run build` again and the timestamp updates to the new build time
- View the source — it's fully static HTML, no JavaScript needed
